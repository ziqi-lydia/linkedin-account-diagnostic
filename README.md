# LinkedIn Account Diagnostic & Growth Prescription

> A Sai skill that performs comprehensive LinkedIn account diagnostics for early-stage creators and delivers personalized growth prescriptions.

## What This Skill Does

Give it any LinkedIn profile. It runs a 5-module diagnostic:

| Module | What It Measures | Output |
|--------|-----------------|--------|
| A. Algorithm Health Score | Is LinkedIn helping or throttling you? | Score 0-100 |
| B. Content Strategy Audit | Content mix, hooks, posting rhythm | Scorecard 0-50 |
| C. Engagement Network | Who engages, network density | Network health map |
| D. Profile-Content Alignment | Does profile match content? | Alignment score |
| E. Growth Prescription | Action plan + inner circle | Checklist + names |

## The Problem

Most early-stage LinkedIn creators hit the same wall:
- Post great content but get zero engagement
- Don't understand why reach is declining
- Don't know what LinkedIn's algorithm rewards
- Don't know WHO to connect with

This skill diagnoses all of it with data.

## Key Concepts

### Algorithm Health Score (0-100)
5 factors, each 0-20 points: Engagement Rate, Posting Consistency, Content Mix, Format Variety, Link Penalty.

### Death Spiral Detection
Checks: zero-reaction rate > 50%, impressions declining > 50%, most recent post < 200 impressions.

### The 4:2:1 Content Rule
- Value/Insight: 57% - Frameworks, lessons, how-tos
- Story/Learning: 29% - Personal experiences, building journey
- Product/CTA: 14% - Product solving a real problem

### Inner Circle Builder
Searches LinkedIn for a personalized 3-tier engagement circle:
- Tier 1 (8-10): Core supporters. Engage with every post.
- Tier 2 (10-15): Slightly bigger accounts. Engage 2-3x/week.
- Tier 3 (20-30): Industry voices. Engage occasionally.

## How to Use

In Sai: "Diagnose my LinkedIn account: linkedin.com/in/your-profile"

## Safety
- Read-only: Never posts, comments, or likes
- Rate-limited: Random delays between page loads
- CAPTCHA-aware: Stops if verification appears
- Max 40 profile visits per session
